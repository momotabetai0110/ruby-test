【CRUD機能】

bin/rails generate model task title:string description:string

マイグレーション実行
bin/rails db:migrate

コントローラー作成
bin/rails generate controller Tasks index new show edit

コントローラー作成
bin/rails generate controller Api::Tasks index show create update destroy
